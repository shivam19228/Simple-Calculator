# Modern Web Calculator

A professional, responsive web application developed using HTML5, CSS3, Bootstrap 5, and JavaScript. This project is designed to demonstrate core front-end development skills, responsive layouts, and DOM manipulation.

## Technical Features
* **Responsive Layout:** Fully compatible with mobile,and desktop screens using Bootstrap's grid system.
* **User Interface:** A high-contrast dark theme designed with custom CSS for better accessibility.
* **Core Logic:** Robust mathematical processing for basic arithmetic operations (Addition, Subtraction, Multiplication, Division).
* **Exception Handling:** Integrated validation to manage and reset invalid mathematical expressions.
* **Utility Functions:** Includes 'AC' for full reset and 'DEL' for character-by-character deletion.

## Technology Stack
* **Frontend:** HTML5 (Semantic Structure)
* **Styling:** CSS3 and Bootstrap 5 (Framework)
* **Scripting:** JavaScript ES6+ (Logic and Event Handling)
