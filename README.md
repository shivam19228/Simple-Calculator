# Simple-Calculator
Simple Calculator using HTML, CSS and JavaScript

A modern, responsive, and professional web calculator built using HTML5, CSS3, Bootstrap 5, and JavaScript.  
This project demonstrates strong front-end development skills including responsive design, DOM manipulation, and clean UI implementation.

## ✨ Features

- Responsive Design (Mobile,Desktop)
- Basic Arithmetic Operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- AC (All Clear) Button
- DEL (Delete Last Digit)
- Error Handling for Invalid Expressions
- Clean and User-Friendly Interface
- Smooth Button Interaction
